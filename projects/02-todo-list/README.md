# To-Do List

Task manager that runs in the terminal.

## Features

- Add and remove tasks
- Mark tasks complete/incomplete
- Filter by status
- View completion stats
- Clear all tasks (with confirmation)

## Run It
```bash
node todo.js
```

## Data Structure

Each task is an object:
```javascript
{
  id: 1,
  task: "Learn JavaScript",
  completed: false
}
```
