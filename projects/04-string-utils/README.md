# String Utilities

10 functions for working with text.

## Functions

**Transform:** capitalize, reverse, toTitleCase  
**Analyze:** countWords, countVowels, isPalindrome  
**Format:** truncate, removeSpaces, kebabCase  
**Extract:** extractNumbers  

## Examples
```javascript
capitalize("hello")          // "Hello"
isPalindrome("racecar")      // true
countWords("I love code")    // 3
kebabCase("Hello World")     // "hello-world"
```

## Run Tests
