# Number Guessing Game

Try to guess the computer's number. Get hints to help you.

## Features

- 3 difficulty levels (1-50, 1-100, 1-500)
- Hot/cold hints based on distance
- Even/odd hints after 3 tries
- 7 attempts max
- Tracks best score

## Run It
```bash
node number-game.js
```
